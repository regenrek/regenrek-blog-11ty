# regenrek.com Eleventy Blog

My personal blog.

## Under the hood

* [11ty/eleventy](https://github.com/11ty/eleventy) Static site generator
* [webpack](https://webpack.js.org/) 
* [Tailwind CSS](https://tailwindcss.com)
* [postcss](https://postcss.org/)


## Install

```bash
yarn
```

## Dev

```bash
yarn dev
```

## Build

```bash
yarn prod
```

Netlify
```bash
npm run prod
```
